Instrukcja obsługi

Krótki przewodnik po aplikacji Billero.

Szybki start

  • Uzupełnij Moje dane (IBAN, adres, NIP …).
  • Dodaj klientów (firmy lub osoby fizyczne).
  • Utwórz nową fakturę (prostą lub z wieloma pozycjami).
  • PDF możesz udostępniać lub zapisać. Edycja faktury regeneruje PDF.

Otrzymane faktury z AI OCR

Billero potrafi automatycznie odczytać z PDF dostawcę, numer, datę i kwotę za pomocą AI (Anthropic Claude). Funkcja jest opcjonalna i wymaga własnego klucza API.

Uzyskanie klucza API

  1. Odwiedź console.anthropic.com i utwórz bezpłatne konto.
  2. W menu wybierz API Keys → Create Key.
  3. Nazwij klucz (np. Billero) i skopiuj go. Zaczyna się od sk-ant-.
  4. Dodaj metodę płatności w konsoli Anthropic. Odczytanie jednej faktury kosztuje mniej niż 0,001 € (model Claude Haiku).

Konfiguracja w aplikacji

  1. Otwórz Moje dane (zakładka ⚙) → sekcja Klucz API Anthropic (OCR).
  2. Wklej klucz w pole i naciśnij Zapisz.
  3. Klucz jest przechowywany w bezpiecznej części urządzenia — nigdy nie trafia do plików aplikacji.

Kopia zapasowa & przywracanie

Wszystkie dane możesz eksportować i przywracać przez zakładkę Moje dane → sekcja Kopia zapasowa.

  • Naciśnij Eksportuj kopię zapasową — aplikacja tworzy plik ZIP.
  • Zapisz plik w iCloud, Google Drive lub gdziekolwiek indziej.
  • Przywracanie: Moje dane → Przywróć z kopii zapasowej → wybierz ZIP.

Faktury cykliczne

Billero automatycznie generuje regularne faktury (tygodniowo, miesięcznie lub kwartalnie) przy każdym uruchomieniu aplikacji.

  1. W zakładce Faktury naciśnij ikonę ↻ (prawy górny róg) → Faktury cykliczne.
  2. Naciśnij + i wypełnij: nazwa szablonu, klient, opis, kwota, stawka VAT.
  3. Wybierz częstotliwość: tygodniowo, miesięcznie lub kwartalnie.
  4. Dla miesięcznych/kwartalnych wybierz dzień miesiąca (1–28).

Płatności krypto

Billero obsługuje dwa typy płatności krypto: statyczne kody QR na fakturze PDF i link płatności Coingate dla 70+ kryptowalut.

Kody QR na fakturze PDF

Ustaw adresy portfeli w Moje dane → Płatność krypto. Kod QR automatycznie trafia na stronę 2 każdej faktury PDF.

  • EURC — 1:1 EUR · Ethereum, Base, Solana
  • USDC — kurs ECB EUR/USD · Ethereum, Base, Polygon, Solana
  • BTC — Bitcoin BIP-21 · kurs CoinGecko
  • ETH — Ethereum native EIP-681 · kurs CoinGecko
  • SOL — Solana native · kurs CoinGecko
  • Własny token — podaj nazwę i adres; QR koduje adres

Kurs i czas pobrania są wydrukowane pod każdym kodem QR. Bez internetu kod QR nie jest wyświetlany.

Coingate payment link

  1. Utwórz konto na coingate.com i uzyskaj klucz API.
  2. W Moje dane → Coingate wprowadź klucz API.
  3. W zakładce Faktury naciśnij ikonę ₿ przy fakturze.
  4. Udostępnij link klientowi — klient płaci dowolną kryptowalutą.
  5. Ty otrzymujesz EUR na rachunek bankowy przez SEPA (Coingate dokonuje konwersji).
