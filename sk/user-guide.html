Návod na použitie

Stručný sprievodca používaním aplikácie Billero.

Rýchly štart

  • Vyplňte Moje údaje (IBAN, adresa, daňové číslo …).
  • Pridajte zákazníkov (firmy alebo fyzické osoby).
  • Vytvorte novú faktúru (jednoduchú alebo s viacerými položkami).
  • PDF môžete zdieľať alebo uložiť. Úprava faktúry regeneruje PDF.

Prijaté faktúry s AI OCR

Billero dokáže z PDF automaticky prečítať dodávateľa, číslo, dátum a sumu pomocou AI (Anthropic Claude). Funkcia je voliteľná a vyžaduje vlastný API kľúč.

Získanie API kľúča

  1. Navštívte console.anthropic.com a vytvorte bezplatný účet.
  2. V menu vyberte API Keys → Create Key.
  3. Pomenujte kľúč (napr. Billero) a skopírujte ho. Začína s sk-ant-.
  4. Pridajte platobnú metódu v Anthropic konzole. Čítanie jednej faktúry stojí menej ako 0,001 € (model Claude Haiku).

Nastavenie v aplikácii

  1. Otvorte Moje údaje (záložka ⚙) → sekcia Anthropic API kľúč (OCR).
  2. Vložte kľúč do poľa a klepnite na Uložiť.
  3. Kľúč je uložený v bezpečnej časti zariadenia — nikdy nie je v súboroch aplikácie.

Záloha & obnova

Všetky dáta môžete exportovať a obnoviť cez záložku Moje údaje → sekcia Záloha.

  • Klepnite na Exportovať zálohu — aplikácia vytvorí ZIP súbor.
  • Uložte súbor do iCloud, Google Drive alebo kdekoľvek inde.
  • Obnova: Moje údaje → Obnoviť zo zálohy → vyberte ZIP.

Opakované faktúry

Billero automaticky generuje pravidelné faktúry (týždenne, mesačne alebo štvrťročne) pri každom spustení aplikácie.

  1. V záložke Faktúry klepnite na ikonu ↻ (vpravo hore) → Opakované faktúry.
  2. Klepnite na + a vyplňte: názov šablóny, zákazník, popis, suma, sadzba DPH.
  3. Zvoľte frekvenciu: týždenne, mesačne alebo štvrťročne.
  4. Pre mesačné/štvrťročné zvoľte deň v mesiaci (1–28).

Krypto platby

Billero podporuje dva typy krypto platieb: statické QR kódy na PDF faktúre a Coingate payment link pre 70+ kryptomien.

QR kódy na PDF faktúre

Nastavte adresy peňaženiek v Moje údaje → Krypto platba. QR kód sa automaticky pridá na stranu 2 každej PDF faktúry.

  • EURC — 1:1 EUR · Ethereum, Base, Solana
  • USDC — kurz ECB EUR/USD · Ethereum, Base, Polygon, Solana
  • BTC — Bitcoin BIP-21 · kurz CoinGecko
  • ETH — Ethereum native EIP-681 · kurz CoinGecko
  • SOL — Solana native · kurz CoinGecko
  • Vlastný token — zadajte názov a adresu; QR kóduje adresu

Kurz a čas získania sú vytlačené pod každým QR kódom. Bez internetu QR kód nie je zobrazený.

Coingate payment link

  1. Vytvorte účet na coingate.com a získajte API kľúč.
  2. V Moje údaje → Coingate zadajte API kľúč.
  3. V záložke Faktúry klepnite na ₿ ikonu pri faktúre.
  4. Zdieľajte link so zákazníkom — zákazník zaplatí ľubovoľnou kryptomenou.
  5. Vy dostanete EUR na bankový účet cez SEPA (Coingate vykoná konverziu).
