Návod k použití

Stručný průvodce používáním aplikace Billero.

Rychlý start

  • Vyplňte Moje údaje (IBAN, adresa, DIČ …).
  • Přidejte zákazníky (firmy nebo fyzické osoby).
  • Vytvořte novou fakturu (jednoduchou nebo s více položkami).
  • PDF můžete sdílet nebo uložit. Úprava faktury regeneruje PDF.

Přijaté faktury s AI OCR

Billero dokáže z PDF automaticky přečíst dodavatele, číslo, datum a částku pomocí AI (Anthropic Claude). Funkce je volitelná a vyžaduje vlastní API klíč.

Získání API klíče

  1. Navštivte console.anthropic.com a vytvořte bezplatný účet.
  2. V menu vyberte API Keys → Create Key.
  3. Pojmenujte klíč (např. Billero) a zkopírujte ho. Začíná sk-ant-.
  4. Přidejte platební metodu v Anthropic konzoli. Čtení jedné faktury stojí méně než 0,001 € (model Claude Haiku).

Nastavení v aplikaci

  1. Otevřete Moje údaje (záložka ⚙) → sekce Anthropic API klíč (OCR).
  2. Vložte klíč do pole a klepněte na Uložit.
  3. Klíč je uložen v bezpečné části zařízení — nikdy není v souborech aplikace.

Záloha & obnova

Všechna data můžete exportovat a obnovit přes záložku Moje údaje → sekce Záloha.

  • Klepněte na Exportovat zálohu — aplikace vytvoří ZIP soubor.
  • Uložte soubor do iCloud, Google Drive nebo kamkoliv jinam.
  • Obnova: Moje údaje → Obnovit ze zálohy → vyberte ZIP.

Opakované faktury

Billero automaticky generuje pravidelné faktury (týdně, měsíčně nebo čtvrtletně) při každém spuštění aplikace.

  1. V záložce Faktury klepněte na ikonu ↻ (vpravo nahoře) → Opakované faktury.
  2. Klepněte na + a vyplňte: název šablony, zákazník, popis, částka, sazba DPH.
  3. Zvolte frekvenci: týdně, měsíčně nebo čtvrtletně.
  4. Pro měsíční/čtvrtletní zvolte den v měsíci (1–28).

Krypto platby

Billero podporuje dva typy krypto plateb: statické QR kódy na PDF faktuře a Coingate payment link pro 70+ kryptoměn.

QR kódy na PDF faktuře

Nastavte adresy peněženek v Moje údaje → Krypto platba. QR kód se automaticky přidá na stranu 2 každé PDF faktury.

  • EURC — 1:1 EUR · Ethereum, Base, Solana
  • USDC — kurz ECB EUR/USD · Ethereum, Base, Polygon, Solana
  • BTC — Bitcoin BIP-21 · kurz CoinGecko
  • ETH — Ethereum native EIP-681 · kurz CoinGecko
  • SOL — Solana native · kurz CoinGecko
  • Vlastní token — zadejte název a adresu; QR kóduje adresu

Kurz a čas získání jsou vytištěny pod každým QR kódem. Bez internetu QR kód není zobrazen.

Coingate payment link

  1. Vytvořte účet na coingate.com a získejte API klíč.
  2. V Moje údaje → Coingate zadejte API klíč.
  3. V záložce Faktury klepněte na ₿ ikonu u faktury.
  4. Sdílejte link se zákazníkem — zákazník zaplatí libovolnou kryptoměnou.
  5. Vy dostanete EUR na bankovní účet přes SEPA (Coingate provede konverzi).
