Benutzerhandbuch

Eine kurze Anleitung zur Verwendung von Billero.

Schnellstart

  • Füllen Sie Meine Daten aus (IBAN, Adresse, Steuernummer …).
  • Fügen Sie Kunden hinzu (Unternehmen oder Privatpersonen).
  • Erstellen Sie eine neue Rechnung (einfach oder mit mehreren Positionen).
  • Sie können das PDF teilen oder speichern. Das Bearbeiten einer Rechnung erstellt das PDF neu.

Rechnungen an Kunden senden

Über die Kundenkarte können Sie schnell mehrere Rechnungen gleichzeitig auswählen und senden.

  • Öffnen Sie die Registerkarte Kunden und tippen Sie auf einen Kunden.
  • Lang drücken auf eine Rechnung → Auswahlmodus aktivieren.
  • Weitere Rechnungen antippen zum Auswählen oder Abwählen.
  • In der Titelleiste auf ↗ tippen, um die ausgewählten PDFs zu teilen.
  • Zum Beenden des Auswahlmodus auf ✕ in der Titelleiste tippen.

Eingangsrechnungen mit KI-OCR

Billero kann aus einem PDF automatisch Lieferant, Nummer, Datum und Betrag auslesen (Anthropic Claude). Die Funktion ist optional und erfordert einen eigenen API-Schlüssel.

API-Schlüssel erhalten

  1. Besuchen Sie console.anthropic.com und erstellen Sie ein kostenloses Konto.
  2. Im Menü API Keys → Create Key auswählen.
  3. Schlüssel benennen (z. B. Billero) und kopieren. Beginnt mit sk-ant-.
  4. Fügen Sie in der Anthropic-Konsole eine Zahlungsmethode hinzu. Das Lesen einer Rechnung kostet weniger als 0,001 € (Claude Haiku-Modell).

Einrichtung in der App

  1. Öffnen Sie Meine Daten (⚙ Registerkarte) → Abschnitt Anthropic API-Schlüssel (OCR).
  2. Schlüssel in das Feld einfügen und auf Speichern tippen.
  3. Der Schlüssel wird im sicheren Bereich des Geräts gespeichert (Android Keystore) — nie in App-Dateien.

Verwendung

  1. In der Registerkarte Rechnungen auf das ↑-Symbol tippen (PDF importieren) und das PDF einer Eingangsrechnung auswählen.
  2. Ein Formular öffnet sich. Auf ✨ Mit KI analysieren tippen.
  3. Kurz warten — Lieferant, Nummer, Datum und Betrag werden automatisch ausgefüllt.
  4. Überprüfen, ggf. korrigieren, dann auf Speichern tippen.

Ohne API-Schlüssel funktioniert das Formular wie gewohnt — alle Felder manuell ausfüllen. Die KI-Schaltfläche wird nicht angezeigt.

Datensicherung & Wiederherstellung

Alle Daten (Rechnungen, Kunden, Einstellungen und PDFs) können über die Registerkarte Meine Daten → Abschnitt Datensicherung exportiert und wiederhergestellt werden.

Export

  • Auf Datensicherung exportieren tippen.
  • Die App erstellt eine ZIP-Datei (billero_backup_YYYY-MM-DD.zip) und öffnet das Teilen-Menü.
  • Datei in iCloud, Google Drive, E-Mail oder einem anderen Ort speichern.

Wiederherstellung (z. B. nach Gerätewechsel)

  • Billero auf dem neuen Telefon installieren.
  • Meine Daten → Aus Datensicherung wiederherstellen öffnen.
  • Warnung bestätigen (vorhandene Daten werden ersetzt).
  • ZIP-Datei vom Speicherort auswählen.
  • Alle Rechnungen, Kunden, Einstellungen, PDFs, Logo und Unterschrift werden wiederhergestellt.

Achtung: Die Wiederherstellung ersetzt alle vorhandenen Daten. Wir empfehlen, vor der Wiederherstellung eine Datensicherung der aktuellen Daten zu erstellen.

Wiederkehrende Rechnungen

Billero erstellt automatisch regelmäßige Rechnungen (wöchentlich, monatlich oder vierteljährlich) bei jedem App-Start.

  1. In der Registerkarte Rechnungen auf das ↻-Symbol tippen (oben rechts) → Wiederkehrende Rechnungen.
  2. Auf + tippen und ausfüllen: Vorlagenname, Kunde, Leistungsbeschreibung, Betrag (netto), MwSt-Satz.
  3. Häufigkeit wählen: wöchentlich, monatlich oder vierteljährlich.
  4. Für monatlich/vierteljährlich den Tag des Monats wählen (1–28).
  5. Beim nächsten App-Start am oder nach dem geplanten Datum wird die Rechnung automatisch mit PDF erstellt.

Vorlagen können jederzeit mit dem Schalter deaktiviert oder bearbeitet werden. Erstellte Rechnungen erscheinen in der Hauptliste.

Krypto-Zahlungen

Billero unterstützt zwei Arten von Krypto-Zahlungen: statische QR-Codes auf der PDF-Rechnung und einen Coingate-Zahlungslink für 70+ Kryptowährungen.

QR-Codes auf der PDF-Rechnung — unterstützte Kryptowährungen

Wallet-Adressen in Meine Daten → Krypto-Zahlung eintragen. Der QR-Code wird automatisch auf Seite 2 jeder PDF-Rechnung hinzugefügt.

  • EURC — Circle Euro-Stablecoin auf Ethereum, Base oder Solana. Kurs: 1 EURC = 1 EUR.
  • USDC — Circle USD-Stablecoin auf Ethereum, Base, Polygon oder Solana. Kurs: EZB über Frankfurter API bei PDF-Erstellung.
  • BTC — Bitcoin, BIP-21 QR-Format. Kurs: CoinGecko bei PDF-Erstellung.
  • ETH — Ethereum (native), EIP-681 QR-Format. Kurs: CoinGecko.
  • SOL — Solana (native), Solana Pay Format. Kurs: CoinGecko.
  • Eigener Token — Token-Name und Wallet-Adresse eingeben; QR kodiert die Adresse (Betrag nach Vereinbarung).

Kurs und Abrufzeitstempel werden unter jedem QR-Code angezeigt. Ohne Internetverbindung wird kein QR-Code angezeigt (Kurs muss aktuell sein).

Unter Meine Daten → Anzeige auf Rechnung können Sie UPN-QR und Krypto-QR unabhängig aktivieren oder deaktivieren. UPN bleibt auf Seite 1, Krypto-QR ist auf Seite 2.

Coingate-Zahlungslink (BTC, ETH, SOL und 70+ Coins)

  1. Konto bei coingate.com erstellen und API-Schlüssel holen.
  2. In Meine Daten → Coingate den API-Schlüssel eintragen. Für Tests den Sandbox-Modus aktivieren.
  3. In der Registerkarte Rechnungen auf das ₿-Symbol neben der Rechnung tippen.
  4. Die App erstellt einen Zahlungslink — QR-Code und Schaltfläche Im Browser öffnen erscheinen.
  5. Link an den Kunden weitergeben. Er zahlt mit einer beliebigen Kryptowährung.
  6. Sie erhalten EUR auf Ihr Bankkonto per SEPA (Coingate übernimmt die Konvertierung).
  7. Zahlungsstatus mit der Schaltfläche Status prüfen in der App überprüfen.

Coingate ist EU-reguliert (EMI-Lizenz, Litauen) und unterstützt SEPA-Auszahlungen.
